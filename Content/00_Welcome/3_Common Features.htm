Top Navigation Options

The top navigation options allow the user to Navigate through the FormFast Connect Main screen. (This screen is customizable. The menu options available across the top of the screen will vary based on this customization.) More information about each of these screens will follow under their respective sections. See the Care Coordinator View section titled Menus to start.

 

App Menu: The App Menu cube opens the App Launcher screen. This screen allows the user to search for apps, offers the option to select, view and navigate All Apps, All Items and access to the Salesforce App Exchange.

FormFast: The FormFast icon option opens the same thing as above. Submitting question.

Home tab: Selecting the Home tab navigates to the Main screen. Options on this screen are described here.

Additional possible tab headings might inlude any of the following:

Common Features for most of these tabs include:

Recently Viewed / All toggle: the options under this toggle will vary depending on which menu you are viewing. Please see these specific Menus options under the Care Coordinator topic.

 

New button: launches the New item screen where a new item can be added. This button will launch a pop up screen specific to the top selected. Please see these specific Menus options under the Care Coordinator topic.

Import button: launches the Import screen allowing you to update up to 50,000 records at a time.

List View Controls buttons: The List View Control menu will have various options available depending on which tab is selected.

Refresh button: Refreshes the current screen.

Edit button: Opens inline editing for specific tab options only.

Chart button: Opens the chart window which enables you to create a new chart.

Show Filters button: Allows the user to view available filters, add a new filter with the New Filter pop up window, add filter logic and remove filters.

 

Main Screen Options

Main screen options allow the user to navigate and adjust settings as needed.

 

Create Button: The create option does nothing right now

 

Help & Training Button: Help and Training options include: Welcome to the Lightning Experience, Set Up Lightning Experience Home, Home Permissions and Settings, What is the Performance Chart, What is the Assistant, Getting Start with Lightning Experience, Find Your Way Around Lightning Experience, Search Help and Training, Get Support, Give Feedback about Salesforce and access to the most current edition of the software Release Notes.

 

Setup: The Setup option allows a Home control panel and a Developer Console as well as the option to Edit the home page.

Search Box

The search box allows the user to search the entire system by entering data or partial data into the search box and clicking enter.

 

 

Other available tabs include:

  • Chatter tab: The Chatter tab, like Facebook for business, allows for a running post log and polls, allows you to follow people and groups, shows who you are following and who is following you, allows you to create groups, edit groups, sort how and what you follow and more. See the section Chatter Menu for more about using this feature.
  • Accounts tab: The Accounts tab contains Account specific information for all of the Common Features described above. See the section Accounts Menu for more about using this feature.
  • Contacts tab: The Contacts tab contains Contacts specific information for all of the Common Features described above. See the section Contacts Menu for more about using this feature.
  • CONNECT Dashboard tab: The Connect Dashboard tab contains the same information as the Home screen described above. See the section Connect Dashboard Menu for more about using this feature.
  • Care Events tab: The Care Events tab contains Care Events specific information for all of the Common Features described above. See the section Care Events Menu for more about using this feature.
  • Care Checklists tab: Opens a pop up window to create a new Care Checklist. See the section Care Checklists Menu for more about using this feature.
  • Care Steps tab: The Care Steps tab contains Care Steps specific information for all of the Common Features described above. See the section Care Steps Menu for more about using this feature.
  • Forms tab: The Forms tab contains Form specific information for all of the Common Features described above. See the section Forms Menu for more about using this feature.
  • Form Templates tab: The Forms Templates tab contains Form Template specific information for all of the Common Features described above. See the section Form Templates Menu for more about using this feature.
  • FormFast Settings tab: This feature contains many available features: FormFast Settings, Notification Batch Settings, Patient Profile Settings and Provider Profile Settings. See the sectionFormFast Settings Menu for more about using this feature.
  • FormSync tab: This option has a help feature and the option to click the sync button to refresh the available form templates provided by FormFast. See the section FormSync Menu for more about using this feature.
  • Reports tab: This option allows you to create new reports, view existing reports and edit reports and folders. See the sectionReports Menu for more about using this feature.

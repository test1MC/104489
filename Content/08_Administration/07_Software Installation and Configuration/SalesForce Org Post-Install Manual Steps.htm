SalesForce Org Post-Install Manual Steps

1. Login to the target org

2. Change default Contact page layout to "FormFast"

SETUP > Customize > Page Layouts > Page Layout Assignment

Edit Assignment and set all profiles to “FormFast Contact Layout”

3. Create a new permission set “FormFast User Local” Add Assigned Connected

App “Form Fast”

4. Assign permission sets to users as needed

Admins - FormFast Admin

Users - FormFast User

Users - FormFast User Local

 

Users should be assigned both User permission sets

Admin users should also be assigned Admin permission set

5. Change the OAuth policy for form_fast connected app

SETUP > Administration Setup > Manage Apps > Connected Apps

Edit “Form Fast” connected app

Change Oauth Policies > Permitted Users to “Admin approved users are pre-authorized”

6. Add whitelist for origin point (FF Server).

SETUP > Administration Setup > Security Controls > CORS

Click New

Enter address of server. For FF test server use https://52.26.20.56

 

Without this step, archiving a document (with attachment) will fail

Installing the Package

To Install the FormFast Connect package in the target org, complete the following steps:

1. Login to the target org

2 .Paste the package link into the browser

3 .When package displays, click CONTINUE

4. Approve third-party access. Check YEW and click CONTINUE

5. Approve package API Access. Click NEXT

6. Choose security level. Select “Grant access to admins only” (default). Click NEXT.

7. Install Package. Click INSTALL

8. Package Install Complete.

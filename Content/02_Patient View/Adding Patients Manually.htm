Manually Add a Patient

When manually adding a patient to FormFast Connect you must first create an Account, then create a Contact.

Create an Account

How to Create an Account

The minimum requirements for adding an account are:

  • Enter the First Name
  • Enter the Last Name

You can also add any other fields required by your organization, per your SalesForce setup.

The intention is to automate this step in the future.

 

Create a Contact

How to Create a Contact

An account must be created first.

When manually creating an account, enter the following Contact fields:

The minimum requirements for adding a Contact are:

  • Enter the First Name
  • Enter the Last Name
  • Select the Account Name (see above)
  • Enter Birth Date
  • Enter Telephone Number
  • Enter Mailing Address
  • Click Save

You can also add any other fields required by your organization, per your SalesForce setup.

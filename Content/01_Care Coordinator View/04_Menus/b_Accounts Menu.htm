User Details Screen

The User Details screen is divided into four main sections: User Information, User Chatter, User Details and Page Features. These sections are detailed below.

Each of these fields is editable by clicking the EDIT button in the upper-right corner of the screen.

Some of the information in the User Information section consists of blue text links. Text links can be clicked to open additional information about the selected item.

User Photo: This is the photo associated with this User. System generated icons are uploaded until the User or User's manager updates this graphic. (can this be done automatically??)

User Name: The name of the User selected.

Title: The User's job title.

Email: The email address associated with this user. If you click this blue text link, your email client will launch with this email address in the "To" field.

Address: This section is a free form text field. Depending on Company policy this address may be the User's personal address (possibly for payroll or similar purposes) or the location address where this User is employed. Clicking this blue text link will open googlemaps for easy navigation.

About Me: The About Me section is a free form text field. This field can contain any information the User or Employer would like.

Manager: This User's acting Manager. If you click this blue text link, it will open the User Details screen for this User.

Company Name: The name of the organization where this User is employed.

Phone: Depending on Company policy this number may be the User's personal phone number (possibly for Human Resources or similar purposes) or the phone number of the location where this User is employed.

Cell: This could be the User's personal cell phone number or a number associated with their position, depending on Company policy.

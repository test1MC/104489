Forms Menu

The Forms Menu has a view sort option dropdown which determines what the main screen will show.

A user can sort their forms list by selecting one of the below:

All: to view ALL Forms in the system

Approved Forms: Any Form where the "Status" is listed as "Approved" will be displayed

Recently Viewed: Narrows the displayed list of forms to only those most recently viewed

Rejected Forms: Any Form with a "Status" listed as "Rejected" will be displayed

Saved Forms: Any Form with a "Status" listed as "Saved" will be displayed

Submitted Forms: Any Form with a "Status" listed as "Submitted" will be displayed

Once the desired list of forms is showing, several options are available to manage the list:

 

 

 

Use of this menu is described in detail below.

Form Number: The alpha-numeric number assigned to the selected form. This may be the SAME form template over and over again; just given a new number as it is assigned to a new Patient and a new event. This number is event/patient specific and NOT associated with a particular form.
Patient: The Patient for whom this particular form for this particular event was created.
Form Template: The name of the base template that created the particular form
Status: The status of the selected form. These status' make up the sort criteria for the Forms page.
Created Date: This is the date this instance of the form template was created for this patient and this event.
More Dropdown: This dropdown offers two additional choices (see below)

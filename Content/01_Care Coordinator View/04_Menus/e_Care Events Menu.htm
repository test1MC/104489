Care Events Menu

The Care Events menu has a view sort option dropdown which determines what the main screen will show.

Once the desired list of Care Events is showing, several options are available to manage the list:

The main Care Events screen will change depending on the sort list selected. Below is the ALL main screen.

Care Event Name: The Care Event name is a clickable link which, when selected, will redirect the user to the Care Event Details screen. (see below)

More Dropdown: The more dropdown contains two available options:

Care Events Details Screen

The Care Event Details screen provides information about the selected care event as well as several opportunities to edit information about this particular care event.

Care Event Name: The name given to the care event when it was created. This is an editable field.
Type:

The type of this specific event. Choices include: Consultation, Procedure and Surgery. This is an editable field.
Scheduled Date & Time: That date and time the care event is scheduled to occur. This is an editable field.
Assignments  
Patient: This is the name of the patient assigned to this care event. This is a selectable field. See the Contacts Menu section for details on using this screen.
Individual: I have no clue what this does or what the point of it is....
Provider: The Provider of the medical services. This is an editable field.
Provider Practice: The Practice the Provider is associated with. This is an editable field.
Care Coordinator: The Care Coordinator who his in charge of handling this care event. This is an editable field.

 

Care Events Related Screen

The Related screen will provide the user with additional information related to this patient and this care event. Additional information might include: forms for this event, checklists, care steps and other related items.

Care Checklist Name: The Care Checklist names listed under this heading can be selected to present more information. See the Care Checklists Menu section for details on using this screen.
Status: This is the status of the Care Checklist. This may be: Draft, Active, Complete and Canceled.
   
View All: Opens any additional assigned Care Checklists for this care event.
More Dropdown: Is detailed above.

Creating Your First Care Step

Follow the steps below to create a care step:

1. Click "New" under Care Steps

2. For Record Type: select "Instance", click Continue

3. Enter the Care Step Name, this will be displayed for the patient.

4. Select the Type "INSERT LINK TO WORKING WITH CARE STEPS". The choices are: Form, Diagnostic or Self-Care

5. The Care Event Scheduled Date will populate by default. Why? What if I want to change it?

6. If this is a Diagnostic step, check the Comment Required field. Enter the message that should be displayed to instruct the user to enter their comment.

7. The Patient and Provider Practice fields will have an auto-fill default.

8. Enter a brief, instructional description of the step in the Description field.

9. Enter an integer in the Due Threshold field to indicate when the step should be due relative to the care event date.

10. If this is a "Form" step, select a specific form to be completed by the patient in the Form Template field.

11. Leave all other fields as default values.

 

Although additional steps are available for your convenience, the fields above are the minimum required to create a Care Step.

Create a Templated Checklist

Your organization can create a templated Checklist that can be reused for multiple Care Events.

This time-saving step will allow you to create one template which can be reused across an unlimited number of Care Events. For example, your organization could create a "Pre-Surgical" Checklist, and assign it to all patients with similar upcoming procedures.

To designate a Checklist as a template, select the "Template" field rather than the "Instance" field when creating a new Checklist.

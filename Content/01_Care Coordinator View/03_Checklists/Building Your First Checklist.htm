Building Your First Checklist

Specify Checklist Information

1. For the Record Type, select "Instance"

2. Enter a name of your choosing as the Care Checklist Name

3. In the Status field, select "Draft"

4. Select the Care Event. If you are creating the checklist from a Care Event, the care event will fill by default.

Add Care Event Steps:

1. Under Care Steps, Click "New"

2. For Record Type, select "Instance" and then click "Continue"

3. Enter the "Care Step Name"; this will be displayed for the patient.

4. Select the Type (See Working with Care Steps for more information)

a. Form

b. Diagnostic

c. Self-care

5. The Care Event Scheduled Date will populate by default.

6. Check the Comment Required field if the step is the Diagnostic type. Enter the message that should be displayed to prompt and instruct the user to enter this comment.

7. Leave the Patient and Provider Practice fields blank to be filled by default.

8. Enter a brief, instructional description of the step in the Description field.

9. Enter an integer in the Due Threshold field to indicate when the step should be due relative to the care event date.

10. If this is a "Form" type, select a specific form to be completed by the patient in the Form Template field.

11. Leave all other fields as default values.
